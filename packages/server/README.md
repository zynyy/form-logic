# @formlogic/server

## Install 安装

```pnpm
pnpm add @formlogic/server -D
```

```yarn
yarn add @formlogic/server -D
```

```npm
npm install @formlogic/server -D
```

## Usage 使用

### dev

开发服务，保存绘制逻辑流程数据以及可视化操作低代码配置json的文件

```json5
{
  scripts: {
    'run-server': 'formlogic-server dev',
  },
}
```
